# Maintenance Requests

The Funnel Leasing Resident Portal provides a comprehensive maintenance request system that allows residents to easily report issues, track progress, and communicate with the maintenance team.

## Submitting a Maintenance Request

### Creating a New Request
Residents can submit maintenance requests through multiple channels:

- **Online portal** - Submit requests through the resident portal
- **Mobile app** - Use the mobile app for quick submissions
- **Email** - Send requests via email (automatically converted to tickets)
- **Phone** - Call the maintenance hotline

### Request Information
When submitting a request, residents provide:

- **Issue description** - Detailed description of the problem
- **Location** - Specific area or room where the issue occurs
- **Priority level** - Urgency assessment (emergency, urgent, routine)
- **Access information** - When maintenance can access the unit
- **Contact preferences** - Preferred communication method

### Photo and Video Upload
Enhanced request documentation:

- **Photo upload** - Upload multiple photos of the issue
- **Video upload** - Record videos for complex problems
- **File size limits** - Up to 10MB per file, 50MB total
- **Format support** - JPG, PNG, MP4, MOV files
- **Automatic compression** - Optimize file sizes for faster upload

## Request Tracking

### Status Updates
Real-time status tracking:

- **Submitted** - Request received and logged
- **Under Review** - Maintenance team reviewing the issue
- **Scheduled** - Work scheduled with date and time
- **In Progress** - Maintenance work has begun
- **Completed** - Work finished and verified
- **Closed** - Resident confirms completion

### Communication Features
Stay informed throughout the process:

- **Email notifications** - Status change notifications
- **SMS alerts** - Text messages for urgent updates
- **Portal updates** - Real-time status in the portal
- **Comments and notes** - Two-way communication with maintenance team

### Timeline Tracking
Monitor request progress:

- **Submission time** - When the request was submitted
- **Response time** - How quickly the team responded
- **Resolution time** - Total time to complete the work
- **Follow-up time** - Time for resident confirmation

## Priority Levels

### Emergency (Immediate Response)
Issues requiring immediate attention:

- **No heat/AC** - Temperature control failures
- **No water** - Complete water service interruption
- **Electrical issues** - Power outages or safety hazards
- **Security problems** - Lock failures or security breaches
- **Flooding** - Water leaks or flooding

### Urgent (24-48 hours)
Issues requiring prompt attention:

- **Partial HVAC** - Reduced heating or cooling
- **Minor leaks** - Small water leaks
- **Appliance failures** - Non-critical appliance issues
- **Minor electrical** - Outlet or switch problems
- **Lock issues** - Difficult but functional locks

### Routine (3-7 days)
Standard maintenance requests:

- **Cosmetic repairs** - Paint touch-ups, minor damage
- **Appliance maintenance** - Regular service and cleaning
- **Minor plumbing** - Slow drains, dripping faucets
- **General repairs** - Loose handles, minor wear and tear
- **Preventive maintenance** - Regular inspections and service

## Best Practices

### For Residents
- Provide detailed descriptions of issues
- Include photos when possible
- Be specific about location and timing
- Respond promptly to maintenance team communications
- Confirm completion of work

### For Maintenance Teams
- Respond quickly to emergency requests
- Provide clear status updates
- Communicate scheduling changes promptly
- Document work completed
- Follow up to ensure satisfaction

### For Property Managers
- Monitor request volumes and trends
- Track response and resolution times
- Review resident satisfaction scores
- Identify opportunities for improvement
- Maintain adequate staffing levels
