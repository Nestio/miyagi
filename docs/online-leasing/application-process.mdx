# Application Process

The Funnel Leasing Online Leasing platform streamlines the entire application process, making it easy for applicants to complete their lease applications while providing property managers with comprehensive information and tools.

## Application Workflow

### Step 1: Initial Inquiry
The application process begins when a prospect shows interest in a property:

- **Property selection** - Choose from available units
- **Preliminary questions** - Basic qualification questions
- **Tour scheduling** - Optional property tour
- **Application initiation** - Start the formal application process

### Step 2: Personal Information
Collect comprehensive personal information:

- **Primary applicant** - Full name, contact details, SSN
- **Co-applicants** - Additional applicants if applicable
- **Emergency contacts** - Reliable contact information
- **Current address** - Present living situation
- **Employment history** - Current and previous employers

### Step 3: Financial Information
Gather financial details for qualification:

- **Income verification** - Pay stubs, tax returns, bank statements
- **Employment verification** - Contact information for employers
- **Banking information** - Account details for rent payments
- **Credit authorization** - Permission to run credit checks
- **Additional income** - Other sources of income

### Step 4: Rental History
Document previous rental experience:

- **Current landlord** - Contact information and relationship
- **Previous landlords** - Past rental history
- **Rental references** - Character and payment history
- **Eviction history** - Any past evictions or legal issues
- **Pet information** - If applicable, pet details and deposits

### Step 5: Property-Specific Information
Collect information specific to the property:

- **Move-in date** - Preferred and required move-in timing
- **Lease term** - Desired lease duration
- **Occupants** - All people who will live in the unit
- **Vehicle information** - Cars, parking needs
- **Special requests** - Accommodations or modifications needed

## Application Features

### Save and Resume
Applicants can save their progress and return later:

- **Automatic saving** - Progress saved every 30 seconds
- **Email reminders** - Gentle reminders to complete application
- **Mobile-friendly** - Complete application on any device
- **Offline capability** - Work without internet connection

### Real-time Validation
Immediate feedback on application completeness:

- **Required field validation** - Highlight missing information
- **Format checking** - Verify email, phone, SSN formats
- **Logical validation** - Check for reasonable dates and amounts
- **Completeness scoring** - Show application completion percentage

### Document Upload
Secure document management:

- **Multiple formats** - PDF, JPG, PNG, DOC files
- **Drag and drop** - Easy file upload interface
- **Document preview** - View uploaded files
- **Secure storage** - Encrypted document storage
- **OCR processing** - Automatic text extraction from images

## Application Management

### Property Manager Dashboard
Comprehensive view of all applications:

- **Application status** - Track progress through review process
- **Priority scoring** - Automated ranking based on criteria
- **Document review** - Easy access to uploaded documents
- **Communication tools** - Direct messaging with applicants
- **Approval workflow** - Streamlined approval process

### Application Review Process
Structured review workflow:

1. **Initial screening** - Automated checks for completeness
2. **Document verification** - Review uploaded documents
3. **Background checks** - Credit, criminal, rental history
4. **Income verification** - Confirm employment and income
5. **Final approval** - Manager decision and lease generation

### Communication Tools
Keep applicants informed throughout the process:

- **Status updates** - Automatic notifications of progress
- **Document requests** - Request additional information
- **Approval notifications** - Immediate notification of decisions
- **Next steps guidance** - Clear instructions for next actions

## Integration Features

### CRM Integration
Seamless connection with CRM system:

- **Lead conversion** - Automatic lead to applicant conversion
- **Status synchronization** - Real-time status updates
- **Communication history** - Complete interaction log
- **Performance tracking** - Application conversion metrics

### Background Check Services
Integrated third-party services:

- **Credit reports** - TransUnion, Experian, Equifax
- **Criminal background** - National and local checks
- **Eviction history** - Comprehensive rental history

### Payment Processing
Handle application fees and deposits:

- **Application fees** - Secure payment processing
- **Holding deposits** - Reserve units during application
- **Security deposits** - Process required deposits
- **Multiple payment methods** - Credit cards, ACH, digital wallets

## Best Practices

### Application Completion
- Keep applications concise but comprehensive
- Use clear, simple language
- Provide helpful examples and guidance
- Offer support for applicants who need help

### Document Requirements
- Clearly specify required documents
- Provide examples of acceptable formats
- Offer alternative document options
- Explain why each document is needed

### Communication
- Set clear expectations for timeline
- Provide regular status updates
- Be responsive to applicant questions
- Use multiple communication channels
