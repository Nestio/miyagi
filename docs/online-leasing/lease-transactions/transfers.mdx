# Transfer Workflow

The Transfer workflow in Funnel Leasing Online Leasing provides a comprehensive solution for managing resident transfers between units, ensuring a smooth transition while maintaining proper documentation and compliance.

## Overview

Our digital transfer system streamlines the entire transfer process, from initial request to final lease execution, providing flexibility and automation while maintaining compliance with all applicable regulations and community policies.

## Key Features

### Lease Detail Management
- **The system allows for the addition, removal, and management of lease details**, including roommates, rentable items, and fees during the transfer process.
- **Comprehensive household management** during transfer process
- **Flexible rentable item modifications** based on new unit availability
- **Fee structure adjustments** for the new unit

### Roommate Changes
- **Roommate changes can be made during transfers** and can be configured to adhere to custom screening rules.
- **Flexible household composition** adjustments during transfer
- **Screening requirements** for new household members
- **Role modifications** as needed for the transfer

### Rentable Item and Fee Adjustments
- **Changes to rentable items (e.g., parking, pets) and associated fees can be made as needed.**
- **Real-time inventory checking** for available rentable items in the new unit
- **Automatic fee recalculation** based on new unit specifications
- **Seamless transition** of existing rentable items where possible

### Automated Task Configuration
- **Automated tasks can be set up to trigger upon completion or action events**, ensuring that data management is efficiently transferred to the property management system (PMS).
- **Customizable workflow automation** based on community needs
- **Integration with existing task management systems**
- **Role-based task assignments** and notifications

### New Lease Generation
- **A new lease is automatically generated for e-signature**, including all required addenda.
- **Template-based lease generation** ensures consistency
- **Automatic addenda inclusion** based on transfer details
- **Digital signature workflow** for all parties

### Notice to Vacate (NTV)
- **The system automatically generates a Notice to Vacate (NTV) document for e-signature.**
    - If a roommate change occurs, only the remaining roommates will be required to sign the new lease, while all remaining and departing roommates must sign the NTV.
- **Compliance-driven NTV generation** based on jurisdiction requirements
- **Automatic signature workflow** for all required parties
- **Document archival** and record keeping

## Transfer Process Workflow

### 1. Transfer Initiation
- **Resident or agent initiates transfer request**
- **System validates eligibility** for transfer
- **Unit availability checking** for requested transfer
- **Preliminary approval** and documentation requirements

### 2. Transfer Application Processing
- **Residents complete transfer application** with updated information
- **Screening requirements** are determined based on transfer type
- **Documentation updates** are collected as needed
- **Application review** and approval process

### 3. Unit Selection and Configuration
- **New unit selection** based on availability and preferences
- **Rentable item configuration** for the new unit
- **Fee structure adjustment** for the new unit specifications
- **Move-in date coordination** and scheduling

### 4. Document Generation and Execution
- **New lease is automatically generated** with all required addenda
- **NTV document is generated** for the current unit
- **Digital signature workflow** is initiated for all parties
- **Agent countersignature** completes the process

### 5. System Updates and Coordination
- **PMS integration** updates the property management system
- **Billing adjustments** are automatically applied
- **Communication notifications** are sent to all parties
- **Move coordination** and scheduling
